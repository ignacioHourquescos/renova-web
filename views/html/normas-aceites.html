Normas de lubricantes

API · ACEA · ILSAC · SAE

← Volver al inicio

Las etiquetas de los aceites pueden parecer un código secreto. En esta guía te explicamos qué significan las principales normas y cómo elegir el lubricante correcto para tu vehículo o maquinaria.

API American Petroleum Institute

El estándar más reconocido a nivel mundial. Clasifica los lubricantes según rendimiento y compatibilidad con motores.

  • Nafta: API SP, SN, SM — letras más avanzadas = mayor protección
  • Diésel: API CK-4, CJ-4, CI-4 (camiones, flotas)
  • Ejemplo: API SP/CF = nafta y diésel ligero

ACEA Estándar europeo

Exigido por Volkswagen, BMW, Mercedes, Renault, Fiat. Especificaciones más estrictas en intervalos y protección.

  • A/B — Motores nafta y diésel de pasajeros
  • C — Bajo cenizas (compatible con catalizadores)
  • E — Diésel pesados (camiones, buses)

ILSAC Japonés / Americano

Estándar de Toyota, Honda, Nissan y marcas americanas. Enfocado en eficiencia de combustible y protección del motor.

  • GF-6A y GF-6B (versiones más recientes)
  • Mejor protección contra desgaste y LSPI
  • Buscá el sello en el envase si tu manual lo indica

SAE — Viscosidad del aceite

El número antes de la W (winter) indica el comportamiento en frío; el número después, en caliente.

ClasificaciónUso recomendado
0W-20, 5W-30Motores modernos, clima frío, ahorro de combustible
5W-40, 10W-40Uso general, amplio rango de temperaturas
15W-40, 20W-50Motores diésel, vehículos antiguos, clima cálido

Ejemplo: 5W-40 = fluye bien en frío (5) y mantiene protección en caliente (40).
